# 分析仮説と価値の正当化

## 概要

**リサーチクエスチョン（RQ1）**
> 金利1%上昇時に、東京都のマンション取引価格は何%変動するか？

---

## 1. 背景と問題意識

2020年以降、日本の金融政策は歴史的な超低金利から正常化フェーズへ移行しつつある。
2024年3月には日銀がマイナス金利を解除し、住宅ローン金利が上昇局面に入った。

この局面変化は不動産市場に大きな影響を与えると考えられるが、
**「金利が何%上がると、マンション価格は何%下がるか」** を定量的に示したデータ基盤は
一般には公開されていない。

---

## 2. 分析仮説

### H1: 基本メカニズム仮説（負の相関）

**仮説**
> 金利上昇はマンション価格と負の相関を持つ

**メカニズム**

```
金利上昇
  → 住宅ローン月額返済負担の増加
  → 購買可能価格（Affordability）の低下
  → 実需の購買需要減少
  → 売買成立価格の下落
```

**期待される効果量**
- 金利 +1% → マンション取引価格 **-3〜8%** 程度（海外研究の参考値）
- ただし日本市場特有の硬直性がある可能性

---

### H2: タイムラグ仮説

**仮説**
> 金利変動の価格への波及は即時ではなく、**3〜6ヶ月のラグ**が生じる

**メカニズム**

```
金利変動（月T）
  → 市場参加者の認知・心理変化（1〜2ヶ月）
  → 物件探索〜内覧〜価格交渉（2〜3ヶ月）
  → 売買契約・取引成立（+1ヶ月）
  → 国交省への取引価格の届出・反映（+1〜2ヶ月）
```

**検証方法**
- Cross-correlation（クロス相関）で最大相関を持つラグ月数を特定

---

### H3: 地域差仮説

**仮説**
> 都心（東京23区内）は金利感応度が低く、郊外・多摩地区は高い

| エリア | 金利感応度の予測 | 理由 |
|--------|----------------|------|
| 都心3区（千代田・中央・港） | 低 | 富裕層・外国人投資家の実需が強く、下方硬直性がある |
| 城南・城西（渋谷・新宿等） | 中 | 実需と投資需要が混在 |
| 郊外・多摩地区 | 高 | 一次取得者（住宅ローン依存）が主な買い手 |

**示唆**
- 金利上昇局面でも都心物件への影響は限定的、郊外物件は価格調整が顕著になる可能性がある

---

### H4: 価格帯差仮説

**仮説**
> 高額物件（6,000万円超）は金利感応度が低い

**メカニズム**

```
高額物件の購入者
  → 自己資金比率が高い（ローン依存度が低い）
  → キャッシュ購入・富裕層・法人の割合が高い
  → 金利上昇の影響を受けにくい
```

| 価格帯 | 主な買い手層 | 金利感応度の予測 |
|--------|------------|----------------|
| 〜3,000万円 | 一次取得・若年層 | 高（フルローンが多い） |
| 3,000〜6,000万円 | 中間層 | 中 |
| 6,000万円超 | 富裕層・投資家 | 低 |

---

### H5: 築年数差仮説

**仮説**
> 築古物件（築11年以上）の方が金利感応度が高い

**メカニズム**

```
築古物件
  → 投資目的（賃貸収益）での取引比率が高い
  → 利回り ＝ 賃料 ÷ 価格 で評価される
  → 金利上昇 → 期待利回り上昇 → 価格を下げないと成立しない
  → 価格下落圧力が強い
```

---

## 3. 分析ユースケースと価値の正当化

### 3-1. 個人の住宅購入者

**課題**
「今が買い時か、金利が上がったら価格は下がるから待った方がいいのか」判断できない

**このデータが提供する価値**
- 「金利が+0.5%上昇した場合、購入検討エリアのマンション価格は過去データから▲X%の傾向」という定量的根拠
- 金利上昇による「ローン返済額の増加分」と「価格下落による取得コスト削減分」を比較し、最適購入タイミングの判断材料を提供

**意思決定への貢献**
> 金利上昇により返済コストが+200万円増えるが、価格下落で▲400万円安くなる局面では、「少し待つ」判断が合理的

---

### 3-2. 不動産投資家

**課題**
金利局面に応じたポートフォリオ戦略（取得・保有・売却）の根拠が数値化されていない

**このデータが提供する価値**
- 金利上昇局面での地域別・価格帯別の価格下落率を過去実績から把握
- H3（地域差）× H4（価格帯差）を掛け合わせることで、「金利上昇耐性の高い物件属性」を特定

**意思決定への貢献**
> 金利正常化フェーズでは都心高額物件の相対的優位性が高まる、という投資戦略の定量的根拠

---

### 3-3. 不動産仲介業者・デベロッパー

**課題**
金融環境の変化に応じた適切な価格設定・販売戦略の立案が感覚的になっている

**このデータが提供する価値**
- 金利変動とエリア別成約価格の相関係数を参照することで、価格設定の根拠を数値化
- H2（タイムラグ）を活用し、金利変動後の価格調整タイミングを先読みした在庫調整が可能

---

### 3-4. 金融機関（住宅ローン担当）

**課題**
担保評価（LTV: Loan-to-Value）において、金利上昇リスクを定量的に織り込めていない

**このデータが提供する価値**
- 金利シナリオ別の担保価値下落率をストレステストに組み込む根拠データ
- 特にH3（地域差）は、「郊外物件 vs 都心物件」のリスク格差を定量的に裏付ける

---

### 3-5. 政策立案者・研究者

**課題**
日銀の金融政策が実体経済（住宅市場）にどう波及するかを測定するデータが不足

**このデータが提供する価値**
- 金融政策の伝達メカニズム（金利 → 不動産価格 → 家計資産 → 消費）の定量化
- H2（タイムラグ）の実証は、政策効果の発現タイミングの把握に直結

---

## 4. 仮説と分析手法のマッピング

| 仮説 | 分析手法 | 必要なデータ | Gold層での対応 |
|------|---------|------------|--------------|
| H1: 負の相関 | Pearson相関係数、線形回帰 | `fact_transactions.price` × `dim_interest.avg_yield` | `fact_transactions` JOIN `dim_interest` |
| H2: タイムラグ | Cross-correlation（CCF） | 月次平均価格の時系列 | `agg_monthly_regional_metrics` |
| H3: 地域差 | 地域別の回帰係数比較 | `dim_region.region_category` | `dim_region` |
| H4: 価格帯差 | 価格帯分位別の相関分析 | `fact_transactions.price` | `agg_monthly_regional_metrics.count_low/mid/high` |
| H5: 築年数差 | 築年数カテゴリ別の回帰 | `dim_property.age_category` | `dim_property` |

## 5. 期待されるアウトプット

### Power BIダッシュボードで示す主要指標

1. **金利×価格の散布図（時系列アニメーション）**
   - 横軸: 10年国債利回り（月次平均）
   - 縦軸: マンション平均取引価格
   - 色: 年度、サイズ: 取引件数

2. **地域別ヒートマップ**
   - 市区町村ごとの「金利感応度（相関係数）」を色で表現

3. **金利上昇シミュレーター**
   - 入力: 金利変動幅（+0.5%, +1.0%, +1.5%）
   - 出力: エリア別・価格帯別の予測価格変動率

## 6. 分析の限界と留意点

| 限界 | 内容 | 対応方針 |
|------|------|---------|
| 因果関係 | 相関は因果ではない。景気・人口動態等の交絡因子が存在する | 分析結果に免責事項を明記 |
| データ期間 | 2020-2024年は超低金利〜正常化の1サイクルのみ | 限られた期間での分析であることを明示 |
| 四半期粒度 | 国交省APIは四半期単位のため月次の精度に限界 | タイムラグ分析は四半期単位で実施 |
| 東京都限定 | 本データは東京都のみ | 全国展開は追加データ取得が必要 |
